# DynamicIslandBar 1.0.0 发布检查

更新时间：2026-07-13

## 当前已完成

- Release 构建与 343 项自动化测试通过。
- `win-x64` 自包含和依赖框架两种单文件包均可生成，且输出目录互不覆盖。
- 发布清单记录 EXE/ZIP 的 SHA-256、签名状态和运行时依赖。
- 默认山水资源已压缩，发布仅携带简体中文、繁体中文和英文框架资源。
- 卸载强制结束进程后会再次恢复 Windows 任务栏。
- 诊断日志、单实例、开机自启、配置导入导出和退出确认已具备。

## 建议发布包

- 面向普通用户：`self-contained`，无需提前安装 .NET 8 桌面运行时。
- 面向熟悉 .NET 的用户：`framework-dependent`，下载更小，但必须安装 x64 Microsoft Windows Desktop Runtime 8.x。
- 不启用 `PublishTrimmed`：WPF、XAML 和反射场景的裁剪风险高于本次可获得的体积收益。

## 构建命令

```powershell
# 推荐给普通用户
.\scripts\Build-Release.ps1 -Version 1.0.0 -SelfContained

# 小体积包，需要用户安装 .NET 8 Desktop Runtime x64
.\scripts\Build-Release.ps1 -Version 1.0.0
```

生成后以各输出目录中的 `release-manifest.json` 为准核对哈希、依赖和签名状态。

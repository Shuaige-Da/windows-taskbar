# 新功能开发计划

## 功能清单

### 1. 歌词水平弹幕效果 + 修复播放控制按钮

**现状**：歌词使用水平marquee滚动（TranslateTransform动画），播放/暂停按钮存在，音量按钮已实现per-app音量控制。

**改动**：
- 歌词改为水平弹幕风格：多行歌词同时从右往左滑动，像B站弹幕
- 修改`CenterCardLyricsLayer`的XAML布局为多行弹幕容器
- 每句歌词独立动画，错开启动时间
- 修改`LyricsFastTimer_Tick`驱动弹幕位置更新
- 验证播放/暂停、上一首/下一首按钮功能
- 验证音量按钮控制的是音乐app音量（已有`AudioService.SetAppVolume`实现）

**修改文件**：
- `MainWindow.xaml`：修改歌词层布局为弹幕容器
- `MainWindow.xaml.cs`：修改歌词动画为多行水平弹幕

### 2. 胶囊自由拖动 + 四边吸附 + 左右90°旋转统一样式

**现状**：胶囊当前主要围绕TopIsland（顶部居中）和BottomTaskbar（底部全宽）工作，左右吸附是单独竖排布局思路，且不能在屏幕任意位置自由停留。

**改动**：
- 新增`Floating`自由停留态，胶囊可停留在屏幕任意位置，不强制贴边
- 拖动过程中实时检测与上/下/左/右四条边的距离，进入阈值时显示吸附预览
- 松手时如果仍在吸附阈值内，则吸附到对应边缘；否则保持当前位置作为自由停留态
- 左右吸附时不再使用独立侧边栏样式，而是沿用完整胶囊结构，继承顶部胶囊参数后整体旋转90°
- 左右吸附后的应用图标保持竖向排列效果，但本质来自完整胶囊旋转，而不是另一套单独UI
- 自由停留态保持拖动停留前的一致参数，不自动切换到顶部参数
- 左右吸附态继承顶部模式参数，确保左右样式与顶部胶囊统一
- 当胶囊已经吸附在左/右边缘时，一旦拖动脱离边缘阈值，立即恢复成自由停留态，并以当前位置继续拖动
- 修改`CapsuleLayoutManager`支持自由态与四边吸附的布局计算、锚点和旋转规则
- 修改`CapsuleConfig`存储自由停留位置、当前吸附边、以及恢复自由态所需的状态信息
- 拖动结束时保存位置与吸附状态到配置，重启后可恢复

**吸附预览**：
- 吸附预览采用“胶囊轮廓 + 边缘流光跑马灯”方案，作为目标落位提示
- 预览层独立于真实胶囊本体，拖动中只显示目标位置/角度/尺寸，不直接替换本体
- 顶部、左侧、右侧边缘预览统一使用“顶部胶囊参数”生成的完整胶囊轮廓
- 左右边缘预览在顶部胶囊轮廓基础上整体旋转90°后贴靠左/右边缘
- 底部边缘预览使用“上一次底部轮廓参数”生成的完整胶囊轮廓
- 左右吸附最终态也继承同一套顶部胶囊参数，保证预览与最终落位一致
- 拖入阈值显示预览，拖出阈值立即隐藏，松手时再决定是否正式吸附

**修改文件**：
- `CapsuleConfigService.cs`：新增LeftDock/RightDock枚举值
- `CapsuleLayoutManager.cs`：新增自由态/四边吸附布局、旋转与锚点计算
- `MainWindow.xaml.cs`：修改拖动逻辑支持自由停留、边缘检测、吸附预览和最终吸附
- `MainWindow.xaml.cs`：修改布局应用逻辑，保证左右吸附沿用完整胶囊并旋转90°

### 3. 应用图标栏收纳菜单增加搜索功能（搜索+点击启动）

**现状**：`OverflowAppsPopup`显示溢出的应用图标。

**改动**：
- 在`OverflowAppsPopup`顶部添加搜索框
- 搜索框输入时过滤本地已安装应用
- 使用Windows注册表枚举`HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall`下的应用
- 搜索结果以图标网格显示
- 点击搜索结果启动对应应用（使用`Process.Start`）

**修改文件**：
- `MainWindow.xaml`：在OverflowAppsPopup中添加搜索框
- `MainWindow.xaml.cs`：实现搜索逻辑
- 新增`LocalAppSearchService.cs`：枚举本地安装应用

### 4. 胶囊10秒无操作自动隐藏

**现状**：有`_capsuleHoverCollapseTimer`（130ms）用于鼠标离开后收缩胶囊高度。

**改动**：
- 新增`_autoHideTimer`（10秒）
- 鼠标进入胶囊或子元素时重置timer
- timer触发时：胶囊淡出（Opacity动画到0.1）
- 鼠标进入屏幕底部/顶部区域时：胶囊淡入恢复
- 保持窗口`AllowsTransparency=True`

**修改文件**：
- `MainWindow.xaml.cs`：新增auto-hide timer和淡入淡出逻辑

3. **功能2**（左右吸附）— 修改布局系统
4. **功能3**（搜索功能）— 新增独立模块
